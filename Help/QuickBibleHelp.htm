Overview
QuickBible is designed to create instant presentation slides of scripture.  These slides can be displayed on any monitor that is connected to the system.   If an overhead projector is connected to the system as a monitor then this allows you to display slides to an audience.

QuickBible is composed of 4 main functions.  These are:  1) The main Worksheet and settings along with associated functions, 2) The preview and overhead windows, 3) The color and background configuration dialog, and 4) The search dialog.  Depending on your edition, QuickBible will allow you to display verses in a preview and a main display.  All versions have a preview window, but only the Basic and Full editions have the main display.  The Lite and Personal editions only have the preview display.  This display is perfect for previewing verses for reading or for slide preparation.  A useful configuration of different editions of QuickBible might have the backoffice using the Lite or Personal editions for saving off verse lists for later displaying on the Basic or Full editions of QuickBible .  The Basic edition will allow you to search for words or phrases to help you locate information based on a theme.  Perhaps you want to display different verses about tithing but your audience is tired of hearing the same verse over and over.  You can find all verses related to tithing by searching for 'tithe', 'tithing', or a variant of the word.  All editions except for the Lite version have searching and only the Basic and Full editions have verse by verse settings.

Registration

QuickBible needs to be registered with GravyLabs.  GravyLabs needs to be able to send you correspondance related to new versions of QuickBible.  After registration you will not see then Registration screen again.  QuickBible aids in your registration by providing buttons that can launch your email client or create a registration message that is copied to the clipboard.  When you click Email, QuickBible launches your email client and composes a registration message.  This message is created from your entries, which must be entered exactly as they were when you ordered the QuickBible.  If you do not have an email client configured then you can use the clipboard button.  When you click Clipboard, QuickBible creates a registration message and places it on the clipboard.  You can then paste this message into a webmail email message by pressing CTRL+V.

Verse Management

QuickBible manages verse using the Worksheet and the controls in the Add, Delete, File, and Find It groups.  These features work by adding and deleting verses to and from the Worksheet, loading and saving verses in the Worksheet for later access, and searching the scripture for a word or phrase.

Composite of Verse Management functions

QuickBible displays the current selection of the Worksheet or the Settings.  When multiple verses are selected for display QuickBible will only be able to use the overall verse settings in the Display Settings group.  QuickBible can display the current verse selection in the Preview and Main Display.  When the Preview is turned on QuickBible can quickly update it in four ways.  These are: 1) By clicking the Refresh button, 2) By pressing ALT + F keyboard combination, 3) By using the left and right arrow keys to move through the worksheet, 4) By clicking the verse column in the Settings view.

Selecting Verses

Adding Verses

Verses can be added to the Worksheet in four different manners:  1) Loading a previously saved set of verses, 2) Double clicking an empty area of the Worksheet, 3) Clicking the Verse button or 4) Clicking the Next Verse button.  Initially, the first verse added is Genesis 1:1 from the King James version.  Additional verses are added sequentially based on the current highlighted verse.

Autosizing Verses

When a new Bible is loaded or a font change occurs, QuickBible will automatically determine the font point size necessary to display any single verse in that Bible.  You can choose to accept the new font point size, decline it, or not be asked again. This feature is useful if you wish to keep the font size from changing but have a point size that is guaranteed to fit a verse on the screen.


Deleting Verses

Verses can be deleted from the Worksheet in three ways:  1) Loading a previously saved set of verses to clear the current set, 2) Clicking the All button in the Delete group, or 3) Highlighting one or more verses and clicking the Verse button in the Delete group.



Loading and Saving Verses

To load and save verses to and from the Worksheet, click the Load and Save buttons in the File group.  Saving a group of verses to a file is a great way to preserve settings for displaying these verses again and again.  Verse files can be shared with others who have an edition of QuickBible compatible with yours.  This can be useful in building up a concordance or recording scriptures that were discussed.  Global and verse by verse settings are also recorded in the verse file, preserving the look and feel that best suits your needs.



Worksheet Management

The QuickBible Worksheet is used to list and arrange the verses that will be displayed on the overhead projection system.  The overhead projection system is hooked up to the computer as a primary or secondary monitor.  The QuickBible Worksheet is organized with the columns of Translation, Book, Chapter, From Verse, To Verse, and Scripture.  The Translation column displays which Bible translation is being used.  The Book column displays which book of the Bible is being used.  The Chapter column displays which chapter of the book is being used.  The From Verse and To Verse columns display the beginning and ending verse being used from the chapter.  The Worksheet can be navigated with the arrow keys.  Using the up and down arrows moves the highlighted selection.  Moving the left and right arrows moves the highlighted selection and displays the highlighted verse in the preview window if it is turned on.

Editing the Verses

QuickBible allows editing the Translation, Book, Chapter, From Verse, and To Verse settings to customize the verse that will be displayed.  Verses can be displayed in the Preview window or on the Main window.  The Main window refers to the main display screen which could be an overhead projecter or a second monitor.  There are two types of edits allowed: 1) Rapid entry and 2) Regular entry.  Rapid entry consists of editing the fields with the keyboard and using the Tab key to accept the edit and move to the next field.  Rapid entry also does predictive completion of the translation and book, which means that only the first few letters of these fields need to be typed before QuickBible has predicted the right text.  Rapid entry can be exited by pressing the Enter key, which causes the current edit to be accepted.  While editing a field, pressing Tab moves to the next field and pressing Shift + Tab moves to the previous field.  In regular entry, the possible choices are displayed in a selection list.  Selection lists include: 1) Translations of the Bible, 2) Books of the Bible, 3) Chapters and Verses of the Bible.

Settings Management

The QuickBible  Display Settings are used to customize all verses that will be displayed.  These settings apply to all verses and control the font, size, justification, image, color settings, and slide transitions.  Fonts are changed using a selection list.  The font can be sized automatically or it can be given a static setting.  When fonts are sized automatically, they are bound to a point size that shouldn't be exceeded when they are being fit to the display.  This is very useful for normalizing all the slides in way that keeps the font perspective the same between slides.  Justification occurs with the Center checkbox, and if this is checked then center justification is in effect. Otherwise, left justification is in effect.  The Text and Back buttons control the text and background color of the verse.  The Background button controls the background image or secondary background color to be blended with the primary background color.  This is very useful for creating professional looking slides in which the verse is not overpowered by the background image.  Slide transitions control how slides are shown.  When transitions are on, the slide that the verse is on will fade in and out at a rate that can be chosen on the Image Settings dialog.

The Display Settings Group


The QuickBible verse settings are used to customize verses that will be displayed.  These settings are identical in function to the display settings.  QuickBible verse settings are applied on a verse by verse basis and can be turned on or off.  When turned on, the verse setting is used in the display of the verse.  Verse customization is useful when you want to plan ahead and have different fonts, backgrounds, colors, alignments, and justifications for your verses.  Settings are edited by clicking on the property within the settings list and on the verse of interest.  Verse settings are edited differently depending on the setting that is being edited.  The Use, Auto Size, Center, and Transition Slides column settings have only a "Yes" or "No" choice that is presented in a selection list.  The Background, Text, and Back settings go directly to the dialogs that edit image settings, text, and background colors.  These settings can be changed in the dialog or canceled out and left as they were before editing began.  The Font setting provides a selection list of fonts that are currently installed to the system.  These include True Type, Raster, and Device fonts. The Point Size and Point Max settings are edited by typing in new values for the font point size in the event of autosizing being turned off, or the maximum point size not to exceed in case autosizing is turned on.

Choosing a Font

When you click the arrow to the right of the font you will be presented with the font selection dialog.  This dialog allows you to preview the fonts that are installed to your computer.  You can change the font selection by moving around in the list.  You can change the point size of the font by editing the size.  You can scroll around the sample text area to see how the entire alphabet and numbers will be displayed.  Not all computers have the same fonts installed so if you plan on sharing QuickBible bible files (.bbl) with others you should make sure that they have your fonts installed.

Image and Color Management

The Image Settings dialog allows specific control of the background of the display.  Verses are displayed over background images to achieve a professional slide presentation.  QuickBible supports all major image file formats.  Sometimes a background image will overpower the verse that is being overlaid.  It is important to understand how QuickBible presents verse on a background image.  QuickBible uses a graphical concept known as alphablending.  Alphablending combines two images together using a transparency setting and the end result is that one image will appear as a silouhette on the other.  QuickBible will use the main background color as the silouhette to blend in to the background image using the transparency setting.  Look at the example of the preview below to see how an image and an orangeish color were used to effectively display the verse on the image of the desert cavern.  QuickBible is not limited to blending and image and a solid color.  Two solid colors can also be blended. Blending in QuickBible can also be turned off.   Blending transparency is the percentage of the foreground that does not allow the background to show through.  A transparency percentage of 100 will create an opaque background, while a transparency percentage of 0 will create a transparent background.  A transparency in between will control how the color is blended from full transparency to full opacity.

Display Management

QuickBible allows you to display the verse in the Preview and Main display.  Depending on your needs, multiple copies of QuickBible can be running.  The Preview version is a miniature version of what is displayed on the main, overhead display.  When previewing slides that contain background images, some loss of quality will occur in the slide to accomodate shrinking the image to the Preview display size.  The preview display is resizable, and the larger the Preview display, the more accurately it will reflect what will be displayed on the Main display.  This is especially true of verse autosizing in which the main display will be able to display the verse at a larger point size.  The Preview and Main displays can be turned on and off by pressing the On and Off buttons in the Preview and Main Display groups.

Preview Display group


The main, overhead display group controls which display is used for the slide presentation.  It also broadly controls how verses are displayed, either as the entire list of verses in the Worksheet or as the highlighted verse in the Worksheet.  Additionally, a setting exists for adding a citation from which translation is being used.  It also controls updating the presentation with the currently selected verse and moving to the next verse in the list and updating the presentation.

Searching
To search for a particular verse, click the Search button in the Find It group.  Alternatively, you can press F4 on the keyboard.  Type the word or phrase to search for in the Search phrase box.  The text is searched for by pressing Enter in the Search phrase box or by clicking the Find button.  The search can be narrowed by selecting from the Translation, Book, and Chapter lists.  If nothing is selected from these lists, then all Bible translations are searched by book, chapter, and verse. The search results are listed in the Search Results list.  Clicking a single result displays the verse in the lower right hand side of the Search dialog.  The verse can be added to the main Worksheet by clicking the Add or Add All buttons.  The Add button only adds the selected verse while the Add All button adds the entire search results.
